Projeto: SecureVault

Este é um projeto de landing page e responsiva, desenvolvida inteiramente com HTML, CSS e JavaScript, focada em performance, 
acessibilidade e experiência do usuário. O projeto simula uma aplicação completa com funcionalidades como controle de tema e 
formulários interativos.

Configuração e Execução: Como este é um projeto puramente Front-end, ele não requer instalação de pacotes ou servidor.
Utilize um navegador moderno (Chrome, Firefox, Edge, Safari) ou um editor de código (VS Code, Sublime Text, etc.).
Para abrir basta baixar o arquivo do projeto.
Abra a pasta principal do projeto e inicie, dando um clique duplo no arquivo index.html.

Alternativamente, utilize a funcionalidade "Live Server" (se estiver usando VS Code) para garantir que os módulos JavaScript
carreguem corretamente.
